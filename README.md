<h2 align='center'>📝 ROLES</h2>

```java
1. CS 관련 질문 리스트업 (수시)

2. 매 회차별로 자신이 답변할 질문을 N개씩 고른다.

3. 해당 질문에 대해 학습하고 스터디원들에게 설명할 수 있도록 정리한다.

4. 화요일 디스코드로 질문에 대해서 스터디원들에게 답변 및 설명한다.

5. 설명 이후 이해가 부족한 부분에 대해 질의 응답

6. 노션 질문 페이지에 대한 답변을 등록한다.
```
